# The Dude

My `~/.claude` folder, versioned and published. Clone it, point it at Claude Code, u're running XVC.

## What's inside

- `CLAUDE.md` — the pair's operating manual. Shortcuts, rules, voice, constraints.
- `skills/` — slash commands that give the Dude actual capabilities (await, sync, restart, pomo...)
- `commands/` — quick actions (update, ssh, news...)
- `settings.json` — keybindings, hooks, env

No install script. No framework. Just a folder.

## How to use

```bash
git clone git@github.com:limadelic/dude.git ~/.claude
```

Then open Claude Code. Done.

## XVC

The Dude is the reference implementation of [XVC](plans/xvc.md) — XP practices run ungoverned with AI as ur pair.
